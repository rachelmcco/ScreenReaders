The Basics of VoiceOver

By: Rachel McCoy and Kennedy O’Hanley
VoiceOver Sound Settings:
Start/Stop VoiceOver: Control + Option + F5
Repeat What Was Said: Control + Option + Z
Speech Settings: Control + Option + Command + Shift
Increase or Decrease the Reading Speed: Speech Settings + Right Arrow (until you hear "rate") + Up Arrow or Down Arrow (until you are comfortable with the speed)
Increase or Decrease the Volume: Speech Settings + Right Arrow (until you hear "volume") + Up Arrow or Down Arrow (until you are comfortable with the volume)
Start reading: Control + Option + A
Stop reading: Control
VoiceOver Navigation Settings:
Open the Roter: Control + Option + U
Move to the Next Item: Control + Option + Right Arrow or Left Arrow
Move to the Menu: Control + Option + M
Move to the Notification Center: Control + Option + O
VoiceOver Reading Settings:
Spell a Word: Control + Option + WW
Read next/previous word: Control + Option + Right Arrow or Left Arrow
Read next/previous line: Control + Option + Up Arrow or Down Arrow
Read next/previous sentence: Control + Option + Page Up or Page Down
Read next/previous paragraph: Control + Option + Shift + Page Up or Page Down
Read a row in a table: Control + Option + R
Read a row in a table: Control + Option + R
Read a column in a table: Control + C + C
