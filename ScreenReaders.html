How to most effectively navigate websites using VoiceOver

Site that works well with the screen reader for the learner to use for practice

  1. Click the link to Scopee
  2. Start VoiceOver by clicking Control + Option + F5
  3. Try tabbing through the page by Control + Option + and the Arrows. Observe the big ‘skip to main content’ banner that appears for the first tab.
  4. To navigate to differnt item click Control + Option + Right or Left Arrow
  5. To click an Item Control + Option + space
  6. Repeat the process for other items you want to choose
